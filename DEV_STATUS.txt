■ TECHNICAL CONFIGURATION
================================================================================

  Capacitor Config:
    appId:              com.poopbuddy.app
    appName:            PoopBuddy
    webDir:             www
    SplashScreen:       launchShowDuration=0 (disabled)
    AdMob Android ID:   ca-app-pub-9072876824288260~XXXXXXXXXX

  Server Config:
    Main Server:        server.js (Express + SQLite + Firebase Auth)
    AI Endpoint:        POST /api/ai-analyze (Gemini API 프록시)
    Gemini Model:       gemini-2.0-flash
    API Key:            server/.env의 GEMINI_API_KEY
    Port:               3000 (default)

  Build Commands:
    Sync:    npx cap sync android
    Build:   ./android/gradlew.bat -p android assembleDebug
    APK:     android/app/build/outputs/apk/debug/app-debug.apk
    Install: adb -s <DEVICE_ID> install -r <APK_PATH>
    Server:  node server.js

  Key Files:
    index.html   — Structure, splash screen, settings modal, nav
    index.js     — All application logic (4900+ lines)
    index.css    — Styles and design system
    server.js    — Backend server (SQLite, Firebase Auth, AI Proxy)
    server/.env  — API keys (GEMINI_API_KEY)
    capacitor.config.json — Capacitor settings

  Git:
    Latest commits include all changes above.
