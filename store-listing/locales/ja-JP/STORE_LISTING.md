# PoopBuddy - ペット排便記録

## Short Description
AI写真分析でペットの腸健康スコア、パターン、獣医向けレポートを確認

## Full Description
PoopBuddyは、犬と猫のためのAIサポート付き腸健康トラッカーです。

便の写真を記録し、色や形のサインを確認し、カレンダーで日々のパターンを追跡できます。必要なときは、飼い主や獣医師との相談に使いやすいレポートとして整理できます。

主な機能
- Bristol Typeを参考にしたAI写真分析
- 0-100点の腸健康スコア
- カレンダーと週次トレンドの確認
- 獣医相談に使いやすいPDF/CSVレポート
- 複数ペットのプロフィール管理
- プライバシーを重視した記録管理

PoopBuddyの分析結果は健康管理の参考情報であり、獣医師による診断や治療の代わりではありません。気になるサインが続く場合や症状が深刻に見える場合は、獣医師に相談してください。
